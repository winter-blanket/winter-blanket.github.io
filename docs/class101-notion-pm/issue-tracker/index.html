CLASS101 강의 영상​

이 템플릿에서는 이런 Notion 기능을 사용합니다.
  1. Select, Person, Created by, Status Property를 사용해서 할 일 관리에 필요한 정보를 표시합니다.
  2. Notion Button 기능으로 Database 페이지 생성을 자동화합니다.
